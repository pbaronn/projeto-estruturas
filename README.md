## Como Rodar

### Pré-requisito

- Python 3 instalado

---

### Executando os arquivos

Para rodar qualquer problema, basta executar o arquivo correspondente no terminal.

Exemplo:

```bash
python arrays/problema01.py
```

Cada arquivo já possui casos de teste incluídos no próprio código.  
Ao executar o script, os resultados serão exibidos diretamente no terminal.
